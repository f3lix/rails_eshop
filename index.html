Build Status

How to deploy

$ git clone https://github.com/f3lix/rails_eshop
$ cd rails_eshop
$ vim config/database.yml
# Modify config/database.yml, get your database online
$ bundle install
# Resolve dependencies
$ bundle exec rake db:create
# Create database according to db/schema.rb
$ bundle exec rake db:migrate
# Run the migrations
$ bundle exec rake db:seed
$ bundle exec rake db:populate
# Feed your database with some test data
$ bundle exec rake db:test:prepare
# Make your database ready

How to run

$ rails s
# Start WEBrick on port 3000 by default

How to test

$ bundle exec rake test on your machine
$ bundle exec rake spec
# Run the repec test scripts
